# GIFT Quiz Editor Plus
**GIFT Quiz Editor Plus** was created to help instructors create quizzes with the [GIFT syntax](https://en.wikipedia.org/wiki/GIFT_\(file_format\)).

GIFT is a short, intuitive and normalized syntax used originally in the Moodle community to create quiz questions.
It allows storing quiz questions in a textual format that can be versioned, shared and easily modified.

Several aspects of GIFT are not fully supported, because Google's Quizzes and its API are limited in functionality.

## ✨ Enhanced Version
This repository has been significantly modernized and enhanced by [José Antonio Ahumada](https://github.com/jantoniorah).

### New Features & Improvements:
- **Short Answer auto-grading:** Uses regex text validation inside Google Forms.
- **Custom Points:** Parses GIFT weight percentages and correctly applies point values.
- **Page Breaks:** Add `$CATEGORY: <name>` lines in your GIFT text to automatically split long quizzes into sections.
- **File I/O:** Upload and download `.gift` files directly from the sidebar.
- **Live Preview:** See real-time parsing updates and question summaries.
- **Autosave:** Text is auto-saved locally so you don't lose work.
- **UI Modernization:** Complete redesign to match Google Workspace aesthetics, with jQuery dependencies entirely removed in favor of Vanilla ES6 JS.
- **Security:** Patched XSS vulnerabilities in string processing and removed intrusive API warning messages from the generated forms.

## Developer info

### How to update the parser

- `npm run-script build` for the GIFT-grammar-PEG.js project.
- copy the `gift-parser-globals.js` and `gift-parser-globalsJS.html` files to this project.


### How to use clasp

- `clasp pull` to get the code from `script.google.com` project space
- `clasp push` to put it back

Note that these commands take ALL THE FILES and there's no intelligent merging like in git.
