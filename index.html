GIFT Quiz Editor Plus

A Google Forms add-on that lets instructors create quizzes using the GIFT syntax — a concise text format from the Moodle community for writing quiz questions that can be versioned, shared, and easily modified.

Features

  • True / False, Multiple Choice, Short Answer, Matching, Numerical, Essay
  • Live question preview as you type
  • Upload and download .gift files
  • Auto-save across sidebar sessions
  • Page breaks via $CATEGORY: lines
