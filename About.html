GIFT Quiz Editor

Create quizzes using the GIFT syntax, a concise text format from the Moodle community for writing quiz questions that can be versioned, shared, and easily modified.

Supported question types:

  • True / False (with feedback)
  • Multiple Choice & Checkboxes
  • Short Answer (auto-graded)
  • Matching (grid)
  • Numerical
  • Essay
  • Descriptions & Page Breaks

What's new in this version

  • Short Answer questions are now auto-graded via text validation
  • Custom point values parsed from GIFT weight percentages
  • $CATEGORY: lines create page breaks to organize long quizzes
  • Upload and download .gift files directly from the sidebar
  • Live question preview — see parsed questions as you type
  • Autosave — your work is preserved if the sidebar closes
  • Removed intrusive warning messages from generated forms
  • Modernized UI and removed jQuery dependency
  • Security fix: XSS vulnerability patched in error reporting
